# 重要涌现：素材展示轴 / Timeline Asset

记录时间：2026-04-06 04:35:00

标签：

- `重要涌现`
- `素材轴`
- `TimelineAsset`
- `统一抽象`
- `夏夏灵感`

## 这次涌现是什么

夏夏刚刚闪出来的这个点非常重要：

```text
不只是手写图，
其他素材也可以挂时间轴。
```

例如：

- 圈圈
- 叉叉
- 下划线
- 箭头
- 高亮块
- 小贴纸
- 音效
- 音乐

它们本质上都不是“额外功能”，
而是：

**时间轴上的素材。**

## 1. 统一原理

以后不一定只做：

- 语音轴
- 板书轴

还可以再长出：

- 素材展示轴

也就是说：

```text
上面是语音
中间是板书
下面是素材
```

而这些素材以后完全可以越来越丰富。

## 2. 为什么这很重要

因为这一步会让系统从：

- 一个个零散小功能

变成：

- 一个统一的时间轴素材系统

也就是说，后面出现新的需求时，
我们不需要每次都重新发明一个新模块，
而是把它放进同一个抽象里。

## 3. 最小统一结构

可以先用这样一个通用结构去理解：

```ts
type TimelineAsset = {
  id: string;
  track: 'speech' | 'board' | 'overlay' | 'music';
  kind: 'handwrite' | 'circle' | 'cross' | 'underline' | 'arrow' | 'audio';
  startTime: number;
  endTime: number;
  x?: number;
  y?: number;
  width?: number;
  height?: number;
  src?: string;
  label?: string;
};
```

## 4. 这一步带来的好处

### 对 UI

- 可以把下面那条新时间轴做成“素材展示轴”

### 对 AgentB

- AgentB 不只管描红分段
- 以后也可以管素材区间

### 对后续扩展

- 圈圈叉叉不是特例
- 音乐也不是特例
- 高亮块也不是特例

它们只是：

**不同轨道上的素材块。**

## 5. 一句话收束

这次不是只冒出一个点子，
而是长出了一条统一原理：

**以后所有可展示、可定位、可定时出现的东西，都可以成为时间轴素材。**
