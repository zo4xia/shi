## 核心概念

- **日历（Calendar）**：日程的容器。每个用户有一个主日历（primary calendar），也可以创建或订阅共享日历。
- **日程（Event）**：日历中的具体事件，包含时间、标题、地点、参会人等信息。支持单次日程和重复日程。
- **参会人（Attendee）**：日程的参与者，可以是飞书用户、会议室资源、外部邮箱地址。
- **忙闲时间（FreeBusy）**：查询用户在指定时间段的忙闲状态，用于会议时间协调。

## 资源关系

```
Calendar (日历)
└── Event (日程)
    ├── Attendee (参会人)
    └── Reminder (提醒)
```
